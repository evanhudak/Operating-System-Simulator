# Operating-System-Simulator
C-based operating system simulator implementing process scheduling, memory management, and interrupt handling across multiple scheduling policies including FCFS, SJF, SRTF, and Round Robin.

This repository contains a C-based operating system simulator developed as part of an operating systems course project. The simulator models core OS behavior including process scheduling, interrupt handling, and memory management across multiple scheduling policies such as FCFS, SJF, SRTF, and Round Robin.

The project emphasizes systems programming, modular design, algorithm implementation, and simulation of low-level operating system workflows.
