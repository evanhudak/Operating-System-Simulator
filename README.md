# Operating-System-Simulator
C-based operating system simulator implementing process scheduling, memory management, and interrupt handling across multiple scheduling policies including FCFS, SJF, SRTF, and Round Robin.

This repository contains a C-based operating system simulator developed through a multi-phase systems programming project. The simulator models process scheduling, interrupt management, and memory handling using algorithms such as FCFS, SJF, SRTF, and Round Robin. The project demonstrates low-level programming, algorithm design, modular implementation, and operating system fundamentals.
